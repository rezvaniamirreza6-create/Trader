# ربات تلگرامی تحلیل تکنیکال

ربات تلگرامی برای تحلیل تکنیکال بازار کریپتو و فارکس، با سیستم لایسنس مدیریتی.

## ⚠️ توجه مهم قبل از استفاده

این ربات یک **ابزار آموزشی و تحلیلی** است، نه یک ماشین کسب سود تضمینی.
اندیکاتورهای تکنیکال (RSI, MACD, MA) صرفاً داده‌های گذشته را پردازش می‌کنند
و **هیچ تضمینی برای آینده‌ی قیمت ندارند**. اگر قصد دارید این ربات را به
دیگران بفروشید، صادق باشید درباره محدودیت‌هایش — تبلیغ کاذب سود قطعی
می‌تواند از نظر قانونی مشکل‌ساز باشد.

## ساختار پروژه

```
forex_bot/
├── bot.py              # فایل اصلی ربات
├── license_manager.py  # مدیریت لایسنس و کاربران
├── analysis.py         # تحلیل تکنیکال
├── requirements.txt    # پکیج‌های مورد نیاز
├── Procfile            # برای اجرا روی Railway
├── .gitignore          # جلوگیری از آپلود فایل‌های حساس
└── data/                # (خودکار ساخته می‌شود) فایل‌های JSON داده‌ها
```

## مرحله ۱: گرفتن توکن جدید ربات (ضروری!)

توکنی که قبلاً جای دیگری share شده ناامن است. حتماً این کار را بکن:

1. به تلگرام برو و با [@BotFather](https://t.me/BotFather) چت کن
2. دستور `/mybots` بزن و ربات قبلی را انتخاب کن
3. `API Token` → `Revoke current token` بزن تا توکن قبلی غیرفعال شود
4. توکن جدید را کپی کن (این را در کد قرار **نمی‌دهیم** — مرحله بعد توضیح می‌دهد)

## مرحله ۲: آپلود در گیت‌هاب

```bash
cd forex_bot
git init
git add .
git commit -m "Initial commit"
git remote add origin <لینک ریپوی گیت‌هاب شما>
git push -u origin main
```

چون فایل `.gitignore` تنظیم شده، فایل‌های `data/` (که اطلاعات کاربران و
لایسنس‌ها رو دارن) آپلود نمی‌شن — این دقیقاً همون چیزیه که می‌خوایم.

## مرحله ۳: تنظیم روی Railway

1. وارد [railway.app](https://railway.app) شو و یک پروژه جدید از روی ریپوی گیت‌هابت بساز
2. به تب **Variables** برو و این مقادیر رو اضافه کن:

| نام متغیر | مقدار |
|---|---|
| `BOT_TOKEN` | توکن جدیدی که از BotFather گرفتی |
| `ADMIN_ID` | `5999371363` |
| `TWELVE_DATA_API_KEY` | (اختیاری) کلید از twelvedata.com برای داده فارکس |

3. Railway به صورت خودکار `requirements.txt` رو نصب و `Procfile` رو اجرا می‌کنه

## نکته درباره داده فارکس

داده‌ی کریپتو (بخش Binance) رایگان و بدون نیاز به کلید کار می‌کند.
برای داده‌ی فارکس واقعی (EUR/USD و غیره) باید:
1. به [twelvedata.com](https://twelvedata.com) بروی
2. یک حساب رایگان بسازی (پلن رایگان محدودیت تعداد درخواست دارد ولی برای شروع کافیست)
3. کلید API را در Railway به عنوان `TWELVE_DATA_API_KEY` اضافه کنی

## استفاده

### دستورات کاربر:
- `/start` — شروع
- `/activate XXXX-XXXX-XXXX-XXXX` — فعال‌سازی لایسنس
- `/analyze` — دریافت تحلیل (نیاز به لایسنس دارد)
- `/myinfo` — وضعیت اشتراک

### دستورات ادمین (فقط برای ADMIN_ID):
- `/genlicense 30` — ساخت لایسنس با ۳۰ روز اعتبار
- `/listlicenses` — لیست همه لایسنس‌ها
- `/revoke <user_id>` — لغو دسترسی یک کاربر

## توسعه‌های پیشنهادی بعدی

- اضافه کردن بک‌تست خودکار قبل از نمایش سیگنال
- نمودار گرافیکی قیمت + اندیکاتورها (با matplotlib و ارسال عکس در تلگرام)
- پشتیبانی از تایم‌فریم‌های مختلف (1h, 4h, 1d) با دکمه انتخاب
- مهاجرت از JSON به SQLite وقتی تعداد کاربران زیاد شد
